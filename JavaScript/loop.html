loop/iterate

반복문은 컴퓨터에게 반복적인 작업을 지시하는 방법

while

        while (조건){
            반복해서 실행할 코드
        }
        // 조건 안에는 boolean 타입이 와야함

조건이 true에서 false가 될 때까지 중괄호 안에 내용을 반복적으로 실행함

무한 루프 : 조건이 계속 true일 때 페이지에서 계속해서 실행되는 것, 컴퓨터에 부담을 주기 때문에 적당한 시점에서 false로 바뀌게 해야함

        var i = 0;
        while(i < 10){
            document.write('i가 10보다 작을 때 까지만 실행됩니다.');
            i = i + 1;
        }

while은 구성요소들이 떨어져있어 안에 다른 코드를 넣으면 문제가 날 확률이 다분함

for

        for(var i = 0; i < 10; i = i + 1){
            document.write('i가 10번 실행됩니다.');
        }

for문은 조건부분에 모든 구성요소를 묶어둠

조건 마지막에는 세미콜론을 쓰지않음

  1. 초기화 : 제일 처음에 1번만 실행;
  2. 반복 조건 : 반복문을 언제까지 돌릴건지;
  3. 반복 부분 : 어떻게 반복할지

i = i + 1;는 i++/++i와 비슷한 의미임

  • i++ : 처음 i 값부터 실행시작
  • ++i : i 값을 먼저 증가시키고 실행시작

반복문의 제어

break

반복작업을 중간에 중단시키고 싶을 때 사용

        for(var i = 0; i < 10; i++){
            if(i === 5) {
                break;
            }
            document.write('i가 5번째에서 멈춥니다.');
        }

continue

break는 반복문을 아예 빠져나오지만 continue는 중지되었다가 다시 위로 올라가서 실행 시작함

        for(var i = 0; i < 10; i++){
            if(i === 5) {
                continue;
            }
            document.write('5번만 없고 10번째까지 다시 실행됩니다.');
        }

반복문 중첩

        for(var i = 0; i < 10; i++){
            for(var j = 0; j < 10; j++){
                document.write('숫자를 나란히 2개 출력하는 예제' + i + j + '
');
            }
        }

자바스크립트에서는 문자열과 숫자를 더하면 문자열이 됨
