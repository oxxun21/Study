배열

연관된 데이터를 모아서 관리하기 위해서 사용하는 데이터 타입

변수가 하나의 데이터를 저장하기 위한 것이라면 배열은 여러 개의 데이터를 하나의 변수에 저장하기 위한 것

        // 변수는 [ ] 안에 담음
        var member = ['생활코딩','js','프엔독학'];

        > member
        ['생활코딩','js','프엔독학'] 출력

배열에 들어있는 각각의 데이터는 '원소, element' 라고 부름
원소들은 저장될 때 고유한 번호를 부여받음 : index

변수명 뒤에 []를 붙이고 index번호를 쓰면 각 요소를 불러올 수 있음 
        // 배열은 반복문과 함께 할 때 유용
        예제 1.
        function get_members(){
            return ['생활코딩','js','프엔독학'];
        }
        members = get_members();

        document.write(members[0].toUpperCase());
        document.write(members[1].toUpperCase());
        document.write(members[2].toUpperCase());

        > 이때 배열이 제거되거나 추가되면 정의되어있지 않아 출력 오류 발생
배열.length : 배열에 담겨있는 원소 개수를 알려줌 
        예제 2.
        function get_members(){
            return ['생활코딩','js','프엔독학'];
        }
        members = get_members();

        for(var i = 0; i < 3; i++){
            document.write(members[i].toUpperCase());
        }

        예제 3.
        function get_members(){
            return ['생활코딩','js','프엔독학','배열추가','배열추가'];
        }
        members = get_members();

        for(var i = 0; i < members.length; i++){
            document.write(members[i].toUpperCase());
        }   
        // 배열이 추가, 제거 되어도 탄력적으로 반복문을 돌릴 수 있음

배열의 제어

배열.length : 배열 원소 크기

배열.push('a'); : 기존 배열 끝에 'a' 원소 추가

배열.concat(['a','b']); : 기존 배열에 'a','b' 배열 연결하여 출력

배열.unshift('z'); : 기존 배열에 'z'를 시작시점에 추가하고, 기존 index도 1씩 늘어남

배열.splice(index, howmany, element1...); : 배열의 특정구간을 추출하거나, 특정구간에 특정 배열을 추가

  • index : 배열에 추가할 특정 배열의 위치를 가르키는 index, 숫자
  • howmany : index에서부터 제거될 원소들의 수
  • element : index와 index+howmany 사이에 추가될 값, 옵션
        var a = ['a','b','c'];

        a.splice(1,1,'x','y');
        > ["a","x","y","c"]

맨 앞 데이터 제거 : 배열.shift();
맨 뒤 데이터 제거 : 배열.pop();

정렬 : 배열.sort(); > 함수를 정의해서 원하는대로 정의할 수 있음 : 배열.sort(funct)
역순정렬 : 배열.reverse();
