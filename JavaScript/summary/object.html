Object

배열과 유사한 역할을 함, 연관되어 있는 데이터를 담아냄
배열과 다른 점은 배열은 인덱스를 자동으로 숫자를 부여하는 반면, 객체는 인덱스를 숫자나 문자로 사용자가 지정할 수 있음

        객체를 만드는 법
        1.
        var grades = {'생활코딩':10,'js':6,'프엔독학':60};

        2.
        var grades = {};
        grades ['생활코딩'] = 10;
        grades ['js'] = 6;
        grades ['프엔독학'] = 60;

        3.
        var grades = new Object();
        grades ['생활코딩'] = 10;
        grades ['js'] = 6;
        grades ['프엔독학'] = 60;
  • ' ' : 인덱스, key
  • 인덱스 뒤 숫자 : 인덱스에 해당하는 값, value (숫자 외 다른 데이터 타입도 가능)

값을 가져올 때 : "변수명['인덱스']","변수명.인덱스"

객체와 반복문

배열은 저장된 데이터들이 순서를 가지고 있음 : 객체는 순서가 없고 key와 value가 쌍을 이뤄 저장

객체를 반복문에 사용할 땐 for in문을 사용 (for in문은 배열에도 사용 가능)

        var grades = {'생활코딩':10,'js':6,'프엔독학':60};

        for (key in grades) {
            console.log(key);
        }

        > 생활코딩, js, 프엔독학 출력

        for (key in grades) {
            console.log(grades[key]);
        }

        > 10, 6, 60 출력

key는 사용자가 이름을 바꿀 수 있음
