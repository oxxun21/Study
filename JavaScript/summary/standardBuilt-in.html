표준 내장 객체의 확장

자바스크립트가 기본적으로 가지고 있는 객체들을 의미

내장 객체가 중요한 이유는 프로그래밍을 하는데 기본적으로 필요한 도구들이기 때문

프로그래밍은 언어와 호스트 환경에 제공하는 기능들을 통해서 새로운 소프트웨어를 만들어내는 것이기 때문에 내장 객체에 대한 이해가 필요

    자바스크립트의 내장 객체
  • Object
  • Function
  • Array
  • String
  • Boolean
  • Number
  • Math
  • Date
  • RegExp

자바스크립트나 호스트환경에서 주어지는 객체를 조합하여 새로운 객체를 만드는 것은 '사용자 정의 객체'이다.

두 가지를 섞어서 하이브리드로 사용도 가능
