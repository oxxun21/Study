상속

객체의 로직을 그대로 물려 받는 또 다른 객체를 만들 수 있는 기능 : 기존의 로직을 수정하고 변경해서 파생된 새로운 객체를 만들 수 있게 해줌

로직의 재활용이 용이해짐

프로토타입

객체의 원형, 객체는 프로퍼티를 가질 수 있는데 prototype이라는 프로퍼티는 그 용도가 약속되어 있음

prototype에 저장된 속성들은 생성자를 통해서 객체가 만들어질 때 그 객체에 연결됨

Super를 상속받고 싶다면 sub에 할당된 객체는 상속받고자 하는 객체를 sub에 넣어야한다.

객체를 상속받을 때는 상속받고자 하는 객체의 prototype을 사용하면 안되고 prototype의 복제본을 객체로 사용해야한다.
