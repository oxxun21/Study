함수

하나의 로직을 재실행 할 수 있도록 하는 것으로 코드의 재사용성을 높여줌

        Function 함수명 ([인자...[인자]]){
            코드
            return 반환값
        }

함수를 정의하면 함수명을 적어서 호출함 : 함수명();

  • 재사용성
  • 유지보수의 용이
  • 가독성 향상

함수의 입력과 출력

return

출력 : 함수 내에서 사용한 return은 return 뒤에 따라오는 값을 함수의 결과로 반환

return 뒤에 값을 출력 값으로 반환하되, 함수를 종료함 (그 뒤에 코드가 있어도 실행 X)
return이 여러개라면 첫번째 return만 실행되고 종료됨

매개변수

입력 : parameter, 함수명 뒤 괄호에 들어가는 값을 말함
매개변수로 인해 출력 부분에서 함수명 뒤 괄호에 들어가는 것은 '인자, argument'라고 함

복수의 인자

        예시 1.
        Function get_arguments (arg1, arg2){
            return arg1 + arg2;
        }
        alert(get_arguments(10,20));
        alert(get_arguments(20,30));

함수 정의 방법

        1.
        numbering = function(){
            i = 0;
            while(i < 10){
                document.write(i);
                i += 1;
            }
        }

        2.
        function numbering(){
            i = 0;
            while(i < 10){
                document.write(i);
                i += 1;
            }
        }

        3.
        (function(){
            i = 0;
            while(i < 10){
                document.write(i);
                i += 1;
            }
        })();       //함수 정의와 호출을 바로 함, 1회성 : '익명함수'
