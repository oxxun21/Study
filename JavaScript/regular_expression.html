정규표현식

정규표현식은 문자열에서 특정한 문자를 찾아내는 도구임
어떠한 텍스트 안에서 정보를 찾아내거나, 패턴에 해당되는 것을 찾아서 다른 텍스트로 치환할 때 사용함

컴파일 단계에선 패턴을 찾고, 실행 단계에선 찾은 패턴(대상)에 대해서 어떠한 작업을 구체적으로 하게 됨

        정규표현식 리터럴
        var pattern = /a/;  - a가 찾고자 하는 대상
        
        정규표현식 객체 생성자
        var pattern = new RegExp('a');  - new RegExp를 이용하여 정규표현식 객체를 생성

정규표현식 메소드 실행

정규표현식을 컴파일해서 객체를 만들었다면 문자열에서 원하는 문자를 찾아야 함

  • RegExp.exec() : 추출
    •             var pattern = /a/;
            pattern.exec('abcde');  // ['a']
            
            var pattern = /a./;
            pattern.exec('abcde');  // ['ab']
            - a뒤에 .을 붙히면 a를 앞에두고 뒤에 1개의 문자를 의미함
            var pattern = /a/;

            pattern.exec('bcdef');  // null
  • RegExp.test() : 정보확인
    •             test는 인자 안에 패턴에 해당되는 문자열이 있으면 true, 없으면 false를 리턴

            console.log(pattern.test('abcdef'));    // true
            console.log(pattern.test('bcdefg'));    // false

문자열 메소드 실행

  • String.match() : exec()와 유사
    •             var pattern = /a/;

            var str = 'abcdef';
            str.match(pattern);     // ['a']

            var str = 'bcefg';
            str.match(pattern);     // null
  • String.replace() : 치환
    •             var str = 'abcdef';
            str.replace(pattern,'A');   //Abcdef

옵션

  • i : 대소문자 구분 X
    •             var xi = /a/;
            "Abcde".match(xi);  // null

            var oi = /a/;
            "Abcde".match(oi);  // ['A']

            > var ig = /a/ig; 로 합칠 수 있음
  • g : 검색된 모든 결과 리턴
    •             var xg = /a/;
            "abcde".match(xg);  // ['a']

            var og = /a/;
            "abcde".match(og);  // ['a','a']
(\w+)\s(\w+) : 문자가 하나 이상, 중간에 공백이 있는 패턴
  • () : 그룹
  • \w : 문자
  • + : 수량자, 앞에 있는 문자가 하나 이상일 때 유효
  • \s : 공백
